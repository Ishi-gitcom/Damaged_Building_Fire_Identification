# Damaged Building & Fire Identification using MATLAB
A computer vision project that detects fire and damaged buildings from aerial/UAV images using MATLAB image processing and machine learning techniques.
# Overview
This project detects
-  Fire
-  Damaged Buildings
from UAV images using MATLAB.

The project performs
- Image preprocessing
- Feature extraction
- Image classification
- Prediction on unseen images

This system can assist disaster management agencies in rapidly identifying affected regions after natural disasters or fire incidents.
## Features
-> Fire Detection
-> Damaged Building Detection
-> MATLAB Implementation
->  UAV Image Testing
->  Binary Classification
->  Image Preprocessing
->  Feature Extraction
->  Model Training
->  Prediction Pipeline

# train_fire_model.m
→ Trains the fire detection model.

# train_damage_model.m
→ Trains the building damage detection model.

# TEST_UAV.m
→ Tests the trained models on UAV images.
